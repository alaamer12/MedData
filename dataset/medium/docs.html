Medium Articles Dataset Documentation

A comprehensive collection of Medium articles with rich metadata for machine learning and data science research.

Status: Published
Last Updated: 2025-01-15
License: MIT

Overview

The Medium Articles Dataset is a comprehensive collection of 500,000+ articles from the Medium publishing platform. It includes full article content along with rich metadata such as author information, engagement metrics (claps, responses), publication details, and more.

This dataset is ideal for researchers working on:

  • Natural Language Processing (NLP)
  • Content Analysis
  • Recommendation Systems
  • Author Attribution
  • Topic Modeling

Dataset Schema

Each article in the dataset includes the following fields:

Field Type Description
id string Unique identifier for the article
url string Original URL of the article on Medium
title string Title of the article
subtitle string Subtitle or description of the article (optional)
content string Full text content of the article
claps integer Number of claps received by the article
responses integer Number of responses/comments on the article
reading_time float Estimated reading time in minutes
publication_date datetime Date and time when the article was published
last_modified_date datetime Date and time when the article was last modified
author object Information about the article's author
publication object Information about the publication (if published in one)
tags array List of tags/topics associated with the article

Usage Examples

Loading the Dataset with Hugging Face


from datasets import load_dataset

# Load the dataset
dataset = load_dataset("Alaamer/medium-articles-posts-with-content")

# Basic exploration
print(f"Dataset size: {len(dataset['train'])} articles")
print(f"Sample article title: {dataset['train'][0]['title']}")
print(f"Sample article author: {dataset['train'][0]['author']['name']}")

Basic Text Analysis


import pandas as pd
from collections import Counter

# Convert to pandas for easier analysis
df = pd.DataFrame(dataset['train'])

# Find most common tags
all_tags = [tag for tags_list in df['tags'] for tag in tags_list]
most_common_tags = Counter(all_tags).most_common(10)
print("Most common tags:", most_common_tags)

# Articles with most engagement
top_articles = df.sort_values(by='claps', ascending=False).head(5)
print("Top articles by claps:")
for _, article in top_articles.iterrows():
    print(f"{article['title']} - {article['claps']} claps")

Technical Details

Data Collection

This dataset was created by crawling publicly available articles from Medium.com during 2023-2024. The collection process respected robots.txt rules and rate limiting to avoid overloading the source servers.

Preprocessing

  • HTML content was parsed to extract clean text and structure
  • Metadata was normalized to ensure consistent formatting
  • Duplicates and low-quality content were removed
  • Special characters and encoding issues were resolved

Dataset Format

The dataset is available in the following formats:

  • Hugging Face Dataset: For seamless integration with ML pipelines
  • Parquet Files: For efficient data storage and querying
  • CSV Files: For maximum compatibility with tools

Ethical Considerations

This dataset contains only publicly available content. However, users should be mindful of the following:

  • Respect copyright and fair use principles when using the content
  • Be aware of potential biases in the collected data
  • Do not use the dataset for generating misleading or deceptive content
  • Consider the privacy implications when analyzing author information
