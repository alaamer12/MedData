Medium Articles Dataset

A comprehensive collection of Medium articles with rich metadata, perfect for NLP and content analysis.

Dataset Features

Feature Type Description
Content
title string Article title
text string Full article content
subTitle string Article subtitle
Metrics
claps int64 Number of claps received
readingTime float64 Estimated reading time in minutes
wordCount int64 Total word count
Author Info
author string Author name
bio string Author biography
usersFollowedByCount int64 Number of followers
Publication
publicationname string Name of the publication
publicationdomain string Publication website
publicationfollowerCount int64 Publication followers
Timestamps
createdDatetime string Creation date and time
updatedDatetime string Last update time

Getting Started

Python Example

from datasets import load_dataset

# Load the dataset
dataset = load_dataset("Alaamer/medium-articles-posts-with-content")

# Access the data
for article in dataset['train']:
    print(f"Title: {article['title']}")
    print(f"Author: {article['author']['name']}")
    print(f"Claps: {article['claps']}")
    print("---")
